# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in DCMS, please report it privately
rather than opening a public issue.

- Email: mariusz.zlotucha@gmail.com
- Response time: within 5 business days

Please include steps to reproduce and, if possible, a proof of concept.
Do not publicly disclose the issue until it has been addressed.

## Scope

This is an early-stage portfolio/demo project. Production security hardening
is tracked in `dcms-platform-security-i-hardening.md`.
